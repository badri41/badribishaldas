  • NumPy

A full convolutional neural network implemented with only Python and NumPy, including convolution, ReLU, pooling, dense layers, softmax, and backpropagation. The project reached 96.36% accuracy on MNIST and was mainly about understanding the machinery rather than outsourcing it to a framework.
